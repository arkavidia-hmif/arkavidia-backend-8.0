# Arkavidia 8.0 Backend

## Tech Stack

- [Golang](https://go.dev/)
- [Gin](https://gin-gonic.com/)

## Setup and Installation

1. Clone repo
```
git clone https://github.com/arkavidia-hmif/arkavidia-backend-8.0
```
2. Run docker
```
docker-compose up -d
```
